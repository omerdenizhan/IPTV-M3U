# Güncel IPTV M3U Paylaşımı
* Free-To-Air (FTA) olarak nitelendirilen internetten ücretsiz olarak izlenebilen uydu yayınları için listeler oluşturup M3U klasöründe paylaşmaktayım.

# Uyarılar
* Proje içerisinde herhangi bir içeriğin yayını tarafımca yapılmamakta olup farklı kaynaklardan alınan içeriklere IPTV oynatıcıların destekleyeceği şekilde dosya içeriği düzenlemesi yapılarak sizlere sunulmuştur.
* Paylaşımlar içerisinde herhangi bir ücretli platforma ait içerikler veya direkt ücretli içerikler bulunmamaktadır.
* Farklı kaynaklardan alınarak derlenen çalışma dosyalarımı ticari bir eyleme dönüştürülmemesi şartıyla paylaşabilirsiniz.

# Son Güncelleme
* 10 Aralık 2025 tarihinde güncelleme yapılmıştır.
